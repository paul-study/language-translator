# Language Translator

A simple, clean language translator web application built with HTML, CSS, and JavaScript.

## Features

- Translate text between 12 popular languages
- Clean, modern user interface
- Swap languages with one click
- Responsive design for mobile and desktop
- No API key required (uses MyMemory Translation API)

## Supported Languages

- English
- Spanish
- French
- German
- Italian
- Portuguese
- Russian
- Chinese
- Japanese
- Korean
- Arabic
- Hindi

## Usage

1. Open `index.html` in your web browser
2. Select the source language (left dropdown)
3. Select the target language (right dropdown)
4. Enter text in the left text area
5. Click "Translate" or press Ctrl+Enter
6. View the translation in the right text area
